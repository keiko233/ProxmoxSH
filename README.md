## 如何使用它?

复制以下内容在你的 PVE Shell 里运行。

```shell
curl https://raw.githubusercontent.com/keiko233/ProxmoxSH/master/reinstall.sh -o /usr/bin/ProxmoxSH && chmod +x /usr/bin/ProxmoxSH
ProxmoxSH
```

首次使用建议输入数字 `0` 下载镜像，如果下次需要使用直接输入 `ProxmoxSH` 即可。
